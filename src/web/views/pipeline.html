操作指南

Jarvis Engine 编排引擎使用步骤

1
启动引擎
jarvis engine start
Claude Code 通过 MCP stdio 自动拉起，无需手动
2
连接会话
每个编辑窗口自动创建独立会话，互不干扰，流水线状态完全隔离
3
完成 Gate 产出
每个 Gate 对应不同的产出要求：文档、测试、审查报告等
4
点击 Gate 行
引擎验证该 Gate 的条件是否满足（产物文件是否存在）
5
切换会话
左侧列表点击不同会话查看各自独立的流水线状态
