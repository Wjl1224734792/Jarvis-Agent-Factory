### G 组：文件上传 API 详情

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/v1/uploads/init` | POST | 提交 `{bizType, filename, contentType, size}`，返回 fileId + 七牛云上传凭证 |
| `https://up-z2.qiniup.com` | POST | 使用七牛云 form 直传（token + key + file） |
| `/api/v1/uploads/complete` | POST | 提交 `{fileId}`，完成上传注册，返回文件记录 |

### 已知限制

1. **图形验证码**：admin 登录需要输入图形验证码，自动化测试通过直接创建会话绕过
2. **管理员 Rich Text Editor**：懒加载 + React 受控组件，直接 DOM 操作不触发 onChange
3. **会话过期**：admin session 有时效限制（2h），长时间不活动需刷新
4. **七牛云依赖**：文件上传依赖七牛云对象存储，需确保七牛云配置正确
